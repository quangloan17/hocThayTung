To run a project, for example "1.firstapp":
 1. Create a blank folder with name "firstapp"
 2. Copy all files from "0.blank_project" into "firstapp"
 3. Copy all files from "1.firstapp" into "firstapp"
 4. Go inside "firstapp", check README.txt for additional setups (if any)
 5. Run command: 
	python manange runserver

To run other projects (2., 3., 4. ...), repeat the above steps.
